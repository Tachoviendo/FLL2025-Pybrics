## Métodos para mover los motores individualmente.
### `.run_angle()`
Este método me pide que le pase por parámetro la velocidad en °/s (grados/segundos) y cuantos grados quiero girar.
```python
motorDerecho.run_angle(360,90) 
```
360 es la velocidad y 90 son los grados que girará el motor

### `.run_time()`
Lo mismo que el anterior solo que remplazamos la cantidad de grados que queremos girar por una cantidad de tiempo en milisegundos:
```python
motorDerecho.run_time(360,1000)  
```
esto hace que el motor gire, durante 1 segundo, a 360 °/s de velocidad.

### `.run_target()`
Nuestro motor funciona como una rosa de los vientos

<img src="https://upload.wikimedia.org/wikipedia/commons/c/ce/Rosa_de_los_vientos_31.svg alt="rosa de los vientos" width="150">

* Norte = 0°
* Este = 90°
* Sur = 180°
* Oeste = 270°

si escribimos 
```python
motorDerecho.run_target(0)
```

Cualquiera sea la posición actual del motor, girará hasta llegar al norte, o sea al 0°.

## Código de ejemplo

```python
#Librerias para que funcione pybrics
from pybricks.hubs import PrimeHub
from pybricks.pupdevices import Motor, ColorSensor, UltrasonicSensor, ForceSensor
from pybricks.parameters import Button, Color, Direction, Port, Side, Stop
from pybricks.robotics import DriveBase
from pybricks.tools import wait, StopWatch

#Definir los perisfericos de nuestro robot (motores y sensores)

#motores de movimiento
ruedaIzquierda = Motor(Port.A, Direction.COUNTERCLOCKWISE)
ruedaDerecha = Motor(Port.E, Direction.CLOCKWISE)

#motores de mecanismos
motorDerecho = Motor(Port.F, Direction.CLOCKWISE)
motorIzquierdo = Motor(Port.B, Direction.COUNTERCLOCKWISE)

#Sensores de luz
sensorIzquierdo = ColorSensor(Port.C)
sensorDerecho = ColorSensor(Port.D)

#Definir base de movimiento
robot = DriveBase(ruedaIzquierda, ruedaDerecha, 56, 114)
robot.use_gyro(True)

#Programación del robot

#mover motor derecho
motorDerecho.run_angle(360,90)
wait(1000)

motorDerecho.run_time(-360,1000)
wait(1000)

motorDerecho.run_target(360,0)

```
